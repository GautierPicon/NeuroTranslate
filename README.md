## Contribute / Run Locally

You will need to have [Bun](https://bun.sh/) installed.

Clone the project

```bash
git clone https://github.com/Gautierpicon/NeuroTranslate
```

Go to the project directory

```bash
cd NeuroTranslate
```

Install dependencies

```bash
bun install
```

Start the server

```bash
bun run dev
```
